# easyRAG

#### Description
简单方便的RAG管理系统  用到了 fastapi tortoise orm，使用pgsql vector 插件来存储向量，采用 AI-ModelScope/bge-large-zh-v1.5 将文本编码为向量表示。提供上传 txt、pdf、csv，支持抓取网页链接，将内容上传到不同的智能体和场景。 提供接口可以应用指定关键词查询匹配度，支持搜索指定智能体 的指定场景
